# Macva Press

## Srpski

Macva Press je full-stack web CMS aplikacija za upravljanje vestima i sadržajem, razvijena kao moderna i brža alternativa klasičnim WordPress rešenjima.

Projekat je započet na zahtev klijenta koji je koristio WordPress CMS, sa ciljem da se istraži jednostavnije, lakše i performansno efikasnije rešenje. Aplikacija je razvijana samostalno i nije uvedena u produkciju kao zamena za postojeći sajt.

### Funkcionalnosti

- Administratorski panel za upravljanje člancima i kategorijama  
- Autentikacija i upravljanje korisnicima  
- Korisnički deo sa pregledom vesti, pretragom i komentarima  
- Reakcije (like) na sadržaj  
- Backend API i rad sa SQL bazom podataka  

### Tehnologije

- Backend: Go  
- Frontend: JavaScript  
- Baza podataka: SQL  
- Kontejnerizacija: Docker  

### Status projekta

Projekat je funkcionalan, ali nije završen do nivoa potpune produkcione zamene za postojeći WordPress sistem.

---

## English

Macva Press is a full-stack web CMS application for managing news and content, developed as a modern and faster alternative to traditional WordPress solutions.

The project was initiated at the request of a client using WordPress CMS, aiming to explore a simpler, lighter, and more performance-efficient solution. The application was developed independently and has not been deployed in production as a replacement for the existing site.

### Features

- Administrator panel for managing articles and categories  
- User authentication and management  
- User-facing side with news browsing, search, and comments  
- Reactions (likes) on content  
- Backend API and interaction with SQL database  

### Technologies

- Backend: Go  
- Frontend: JavaScript  
- Database: SQL  
- Containerization: Docker  

### Project Status

The project is functional, but it has not been completed to the level of a full production-ready replacement for the existing WordPress system.
